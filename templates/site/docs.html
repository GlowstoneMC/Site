Wiki

The wiki contains information for both users and developers. If you're a new user and you're looking to get set up, then the wiki is where you need to go.


Our wiki can be found on GitHub.

JavaDocs

JavaDocs are automatically-generated API documentation for developers wishing to work with Glowstone. You can find links to the JavaDocs for each project below.


Glowkit JavaDocs

Glowkit is our fork of PaperSpigot's fork of the Bukkit API. It provides developer-friendly APIs and allows plugins to target a myriad of Bukkit-supporting servers.

Glowstone JavaDocs

Glowstone is our server software. The JavaDocs are provided here for developers of plugins which require access to server internals, and also for contributors to the main project.
