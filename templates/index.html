What is Ultros?

Ultros is a chat bot designed using Python 3 and AsyncIO — with modularity, ease of use and efficiency in mind. It can provide various services, tools and other features to users of chat networks, in a protocol-agnostic way that allows it to work seamlessly on multiple types of chat network at once.
