## Introduction section

Ultros is a highly modular, chat-focused bot. Its purpose is to provide various services and tools to people using chat networks, in a highly extensible and protocol-agnostic way.

Connecting communities, one squid at a time.

## Epeen section

User-friendly

Ultros is easy to use. YAML-based configuration files make Ultros a breeze to set up, without sacrificing power, speed and feature density. All configuration files are organized into an easy-to-understand folder structure, and have example files provided.

Developer-friendly

Ultros is easy to read. Using the fabulous yapsy library, we were able to come up with a plugin system that won't leave your head spinning. If you know Python, you'll pick it up in no time - and there are plenty of example plugins to help you along the way.

Modular

Ultros is extensible. Ultros' robust plugin system means that extending is no problem. Simply drop your files into the plugin folder, add them to the configuration and off you go. Ultros also has a package manager and contrib repository, to keep extensions together. ##
## How to contribute

Efficient

Ultros is fast. A lot of programmers look down on Python for being slower than other, more popular languages. We beg to differ. All our code is written not only with ease of use in mind, but also efficiency - We think it's fast, and we hope you will too.

Protocol-agnostic

Ultros can talk to anything. Ultros introduces a unique protocol system, which allows developers to easily create their own protocol objects. These objects allow Ultros to work over a wide range of platforms. Want an IRC bot? No problem! Mumble? Sure, why not! Ultros ships with both of these protocols as standard, and more are being developed over time.

## First informational section

Why?

There are a lot of bots out there - so many that it would be impossible to list them all off in one place. While a lot of hard work does go into these bots, we feel that they often have limitations that make working with them troublesome. For example, most bots will only be able to connect to one chat network, will only be able to connect once per instance, and will be difficult to customize due to the choice of language.

Ultros solves these problems. By leveraging the power of the incredibly popular Python language, we are able to produce a bot that challenges all of the above problems. We hope it does this sufficiently for most people's needs.

What is Python?

Python is an object-oriented language designed to make coding fast, fun and efficient, whilst still being entirely cross-platform. More information: Python.org
